* Display strings on tab-bar in Emacs.

* How to Use?
  Turn on ~tab-bar-mode~ and ~tab-bar-display-mode~.
  Then, you can see ~tab-bar-display-before~ formatted ~format-mode-line~
  before tabs, and ~tab-bar-display-after~ after tabs.
